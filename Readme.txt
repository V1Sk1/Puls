Проект основан на принципе фотоплетизмографии, который является неинвазивным методом измерения изменения объема крови в тканях с помощью источника света и фотодетектора.
Поскольку изменение объема крови синхронно с биением сердца, этот метод может использоваться для расчета частоты сердечных сокращений.
